## How it works

Counts upwards from 0 to 9 continously if the counter is enabled. Not as special as planned due to medical reasons and finals.

## How to test

Change the position of the enable button from 0 to 1 and look if the 7-segment display does the work as intendet. Maybe try to press the reset button as well. The second input only enables the dot in the display.

## External hardware

LED display, mouse keyboard
