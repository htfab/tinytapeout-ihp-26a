## How it works

Inputs 1 to 7 drive D input of D flop that are connected directly to the 7 segments display.
Input 0 invert All the LED status.


## How to test

Enable the clock at any speed higher than 10 Hz for better experience. 
Use input to control 7 segments display. 

MAp:
Input 1 - 7 Segment LED A.
Input 2 - 7 Segment LED B.
Input 3 - 7 Segment LED C.
Input 4 - 7 Segment LED D.
Input 5 - 7 Segment LED E.
Input 6 - 7 Segment LED F.
Input 7 - 7 Segment LED G.
Input 0 - Invert all 7 Segment LED status.
